总体处理流程（Raw → Interpretable）

  • 1) 数据导入与质量检查（采样率、时窗、轨迹、触发同步）
  • 2) 时间零点校正（Time-zero correction）
  • 3) 去低频漂移与去背景（Dewow + Background removal）
  • 4) 增益补偿（SEC / AGC）
  • 5) 带通滤波与噪声抑制
  • 6) 速度估计与时深转换
  • 7) 偏移成像（Kirchhoff / RTM）
  • 8) 属性分析与解释增强
输出形态：由“可视信号”走向“可解释结构”；参数需按同一工区统一口径。

方法1：时间零点校正（Time-zero）

  • 原理：首波到达时刻受系统延迟、耦合状态和触发抖动影响；需把“地表对应时刻”对齐到统一零点。
  • 常见实现：
  • 阈值法：t0 = min{t | |x(t)| > η·max|x|}
  • 包络法：e(t)=|x(t)+jH{x(t)}|，在上升沿找 de/dt 峰值点
  • 校正：x_c(t)=x(t+t0) 或 t'=t-t0
  • 对GPR数据效果：
  • 浅层反射归位，层界不再整体“上浮/下沉”
  • 双曲线顶点横向拟合误差下降，利于后续速度反演
实操提示：阈值过低易受噪声触发；过高会错过弱首波，常配合 3–5 道滑动稳健估计。
3/12

方法2：Dewow 与去背景

  • 原理：
  • Dewow 去除慢变分量（天线“wow”、系统温漂、近DC基线）
  • 去背景抑制沿测线近似不变的水平分量（直达波/系统串扰）
  • 公式：
  • Dewow：x_d(t)=x(t)-MA_w(x(t))（或高通 x_d = HPF{x}）
  • 去背景：x_b(i,t)=x(i,t)-mean_i[x(i,t)]
  • 对GPR数据效果：
  • B-scan 水平条纹明显减弱，动态范围更均衡
  • 弱小异常体对比度上升，后续偏移伪像减少
参数建议：w 一般取主周期的 3–10 倍；背景窗口过大可能误删大尺度真实层状反射。
4/12

方法3：增益补偿（SEC / AGC）

  • 原理：电磁波传播存在几何扩散 + 介质吸收，深部反射天然弱；增益用于“可视化均衡”，非物理能量恢复。
  • 公式：
  • SEC：g(t)=t^α 或 g(t)=exp(βt)，y(t)=g(t)x(t)
  • AGC：y(t)=x(t)/(sqrt((1/L)∑_{k=t-L/2}^{t+L/2}x^2(k))+ε)
  • 对GPR数据效果：
  • 深层层位和埋设体反射显著增强
  • 但AGC会同步抬高噪声底，可能削弱真实振幅对比关系
实操提示：先轻度带通再AGC更稳；AGC窗长常取目标双曲线半宽附近，避免“过局部放大”。
5/12

方法4：带通滤波与噪声抑制

  • 原理：目标响应集中在主频附近；低频多为漂移/耦合，高频多为随机噪声与采样毛刺。
  • 公式：Y(f)=H(f)X(f)；典型 H(f) 为 Butterworth/Gaussian 带通窗。
  • 常用去噪：
  • 中值滤波：抑制脉冲噪声、保边能力较好
  • 小波阈值：d' = sign(d)·max(|d|-λ,0)（软阈值）
  • SVD低秩分离：X=L+S+N，背景/目标/噪声分离
  • 对GPR数据效果：
  • 双曲线边缘更连续，层位追踪稳定性提升
  • 过窄带通会削弱目标尖锐边界，过强去噪会“抹平”小目标
图例建议：蓝=原始，橙=滤波后，灰=噪声底（同一幅值归一化）
6/12

方法5：速度估计与时深转换

  • 原理：速度由介电常数控制，深度标定是“时间解释”到“工程量化”的关键。
  • 公式：
  • v = c / sqrt(ε_r)
  • 双曲线走时：t(x)= (2/v)·sqrt(z^2 + (x-x0)^2)
  • 时深转换：z = v·t/2（均匀介质近似）
  • 对GPR数据效果：
  • 时间剖面可映射为深度剖面，埋深/层厚可报告
  • v 偏差会直接引入深度比例误差（如 +10% 速度 ≈ +10% 深度）
说明：ε_r 可由双曲线拟合、CMP/WARR 或土层先验联合估计；分层介质建议分段速度模型。
7/12

方法6：偏移成像（Kirchhoff / RTM）

  • 原理：将绕射/散射能量回传到真实反射位置，实现“去双曲化”和几何聚焦。
  • 公式：
  • Kirchhoff：I(x,z)=∫ s(x_r,t=τ(x,z;x_r))·w(x,z;x_r) dx_r
  • RTM：正传 u_f 与反传 u_b 的零延迟互相关
  • I(x,z)=∫ u_f(x,z,t)·u_b(x,z,t) dt
  • 对GPR数据效果：
  • 点目标由开口双曲线收敛为局部高亮点
  • 倾斜界面和复杂构造成像更接近真实几何
代价与风险：RTM 对速度模型与算力更敏感；速度失配会产生弧形伪像或焦点偏移。
8/12

方法7：属性分析与解释增强

  • 原理：把“波形难直接判读”的信息映射为稳定属性场，强化异常体分割与联判。
  • 公式：
  • 包络：A(t)=|x(t)+jH{x(t)}|
  • RMS：RMS= sqrt((1/N)Σx_k^2)
  • 瞬时相位：φ(t)=atan2(Im(z),Re(z))（辅助界面连续性判断）
  • 对GPR数据效果：
  • 包络突出能量异常，RMS反映局部强反射密度
  • 与深度切片结合后，异常体边界和走向更直观
解释注意：属性增强的是“可见性”而非“唯一性”；建议联合原始振幅+地质先验交叉验证。
