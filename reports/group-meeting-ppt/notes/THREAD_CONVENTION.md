# 线程约定

本线程默认绑定 `reports/group-meeting-ppt/` 作为长期工作区。

## 默认行为

以后你在这个线程里提出以下请求时，我默认直接进入该工作区处理：

- 组会 PPT
- 组会汇报 slides
- 周报汇报 deck
- 把实验结果整理成汇报
- 更新上次那份组会

## 默认输出顺序

1. 先给结构草案
2. 再生成 / 更新 `deck.md`
3. 再导出 HTML
4. 如需要再补 PDF / PPTX / 讲稿备注

## 默认标准

- 面向组会，不写空泛废话
- 每页尽量有图 / 指标 / 结论
- 默认 1280x720 基准
- 默认走 html-presentation 方案
- 版本完成后归档进 `history/`
