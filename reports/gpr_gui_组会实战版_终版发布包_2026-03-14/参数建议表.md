# GPR组会实战版｜参数建议表（A8-NEW-1 基线）

> 来源：`evidence/summary.json` 与本次汇报结论。以下为当前 GUI 基线参数，后续应按样本分层微调。

## 推荐默认链路（A链）
- `set_zero_time`: `new_zero_time = 5.0`
- `dewow`: `window = 41`
- `svd_bg`: `rank = 1`
- `fk_filter`: `angle_low = 12`, `angle_high = 55`, `taper_width = 4`
- `sec_gain`: `gain_min = 1.0`, `gain_max = 4.5`, `power = 1.1`
- `hankel_svd`: `window_length = 80`, `rank = 2`

## 对照链路（B链，仅试验）
- `set_zero_time`: `new_zero_time = 5.0`
- `dewow`: `window = 41`
- `subtracting_average_2D`: 默认
- `fk_filter`: `angle_low = 12`, `angle_high = 55`, `taper_width = 4`
- `agcGain`: `window = 31`
- `hankel_svd`: `window_length = 80`, `rank = 2`

## 调参建议（最小可执行）
- 先锁定前处理（zero-time + dewow），再调背景抑制与增益，避免耦合误判。
- 若层状结构被削弱：
  - 优先降低 AGC 强度或回到 SEC；
  - 检查 F-K 角度窗是否过窄导致有效反射被抑制。
- 若背景条纹仍明显：
  - 保持 `svd_bg rank=1` 起步，必要时小步尝试更高 rank；
  - 与 `subtract_avg` 做并行对照，不直接替换默认链路。

## 风险提示
- AGC 在当前样本有“过均衡”风险，不作为默认。
- `bandpass/wavelet` 尚未在 GUI 暴露，暂不能做频带与小波路径的标准化调优。
