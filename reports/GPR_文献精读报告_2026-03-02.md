## 1. F-K域极角高斯滤波杂波去除 (2024)

### 论文信息
- **标题**: A Clutter Removal Method Based on the F-K Domain for Ground-Penetrating Radar in Complex Scenarios
- **期刊**: MDPI Remote Sensing (2024)
- **URL**: https://www.mdpi.com/2072-4292/16/6/935

### 核心算法步骤

```
输入: 原始B-scan数据 M₀ (x道 × t采样点)
输出: 去杂波后数据 M_after

Step A: 峰值匹配均值减法 (直接耦合波去除)
  1. 取中间列数据进行峰值搜索 (二分法)
  2. 找到直接耦合波峰值时间 t_max
  3. 对每道前 2*t_max 样本点做均值减法
     M(i,j) = M₀(i,j) - mean(M₀(i, 1:2*t_max))

Step B: SVD + k-Means 表面抖动去除
  1. 2D-FFT 变换到 F-K 域: N₀(f,k)
  2. 零频移到中心: N(f,k)
  3. SVD分解: N = U·S·Vᵀ
  4. 提取奇异值数组 S_σ = [σ₁, σ₂, ..., σᵣ]
  5. k-Means聚类 (k=4): 将奇异值分为4类
     - S_G: 地面/墙体一次反射波
     - S_S: 地面起伏杂波
     - S_L: 线性干扰杂波
     - S_T: 目标分量
  6. 将前 a+b 个奇异值置零 (a,b为前两类数量)
  7. 重构: N* = U·S*·Vᵀ

Step C: 边缘高斯锥形极角滤波
  1. 计算杂波角度范围 [θ₁, θ₂]
     (经验值: 10°-65°为阻带)
  2. 构建高斯锥形滤波器:
     f(θ) = α/(2πρ) · exp(-(θ-β)²/(2ρ²))
     其中: ρ=0.1 (标准差), β=(θ₁+θ₂)/2, α=2πρ
  3. 频域相乘: N_after = N* · F
  4. 2D-IFFT 回到时域: M_after
```

### 关键参数
| 参数 | 含义 | 经验值 |
|------|------|--------|
| t_max | 直接耦合波峰值时间 | 2-3 ns |
| k (k-Means) | 聚类数 | 4 |
| [θ₁, θ₂] | 阻带角度范围 | 10°-65° |
| ρ | 高斯标准差 | 0.1 |

### 评估指标
- **SCR** (信杂比): 信号区能量 / 杂波区能量
- **IF** (改善因子): 10·log₁₀(SCR_after / SCR_before) [dB]

### 实验结果 (与基线对比)
| 方法 | SCR | IF (dB) |
|------|-----|---------|
| Mean Subtraction | 低 | 基准 |
| SVD (rank=2) | 中 | +~5 |
| RPCA | 中高 | +~8 |
| F-K Filter (传统) | 中 | +~6 |
| **本文方法** | **高** | **+12.91** |

### 可实现要点
1. 直接耦合波去除可用简单均值减法替代
2. SVD+k-Means 可简化为固定rank截断
3. F-K滤波核心是高斯锥形窗，避免理想滤波的振铃效应
4. 角度范围 [10°, 65°] 可作为默认参数

## 2. 窗口长度优化的Hankel矩阵SVD去噪 (2019)

### 论文信息
- **标题**: Noise Suppression for GPR Data Based on SVD of Window-Length-Optimized Hankel Matrix
- **期刊**: MDPI Sensors (2019)
- **URL**: https://www.mdpi.com/1424-8220/19/17/3807

### 核心算法步骤

```
输入: 二维GPR数据 B ∈ R^(N×L), N=采样点, L=道数
输出: 去噪后数据 B_denoised

对每个 trace (一维数据 X = [x(1), x(2), ..., x(N)]):

1. Hankel矩阵构建 (窗口长度 n):
   A = [x(1)   x(2)   ...  x(n)
        x(2)   x(3)   ...  x(n+1)
        ...    ...    ...  ...
        x(m)   x(m+1) ...  x(N)]
   其中: m = N - n + 1, A ∈ R^(m×n)

2. SVD分解:
   A = U·S·Vᵀ, S = diag(σ₁, σ₂, ..., σᵣ), r=min(m,n)

3. 窗口长度优化 (FRFCM准则):
   P(n) = (1/n · Σ(σᵢ - σ̄)⁴)^(1/4)
   n_opt = argmaxₙ[P(n)]
   
4. 奇异值选择 (差分谱法):
   bᵢ = σᵢ - σᵢ₊₁  (差分谱)
   T = ρ · mean(b)  (阈值, ρ为权重系数)
   找分界点 k₁: bᵢ < T 且 bᵢ₊₁ < T 且 bᵢ₊₂ < T
   
5. 重构去噪Hankel矩阵:
   Aₛ = Σᵢ₌₁ᵏ¹ σᵢ·uᵢ·vᵢᵀ
   
6. 恢复一维数据:
   Xₛ = [Aₛ(1,:), Aₛ(2:m, n)]

重复步骤1-6处理所有道，得到二维去噪数据
```

### 关键参数
| 参数 | 含义 | 经验值/方法 |
|------|------|-------------|
| n | Hankel窗口长度 | 100-700，FRFCM优化 |
| ρ | 阈值权重系数 | 1.0 (默认) |
| k₁ | 有效奇异值数量 | 差分谱自动确定 |

### 实验结果
**合成数据 (点目标，SNR=-5dB)**:
- 最优窗口: n=250
- 奇异值截断: k₁=6
- 输出SNR: 7.45 dB (提升 ~12dB)

**不同窗口长度对比**:
| 窗口n | 输出SNR | 效果 |
|-------|---------|------|
| 100 | 5.43 dB | 有毛刺 |
| **250** | **7.45 dB** | **最优** |
| 400 | 6.66 dB | 残留噪声 |
| 700 | 5.19 dB | 过度平滑 |

**与基线方法对比 (合成数据)**:
| 方法 | SNR | 处理时间 | 内存 |
|------|-----|----------|------|
| 传统SVD (局部能量比) | 较低 | 0.45s | 低 |
| 小波变换 | 中 | 1.32s | 中 |
| **本文方法** | **最高** | **1.77s** | **较高** |

### 可实现要点
1. FRFCM计算虽精确，但可用固定窗口 n≈N/8 作为近似
2. 差分谱阈值法可用固定 k₁=5~8 代替
3. 逐道处理可并行化加速
4. 适合随机噪声和相关噪声两种场景

## 3. 杂波去除技术综述 (2024)

### 论文信息
- **标题**: Enhancing subsurface exploration: A comprehensive review of advanced clutter removal techniques for GPR imaging
- **期刊**: ScienceDirect Measurement (2024)
- **URL**: https://www.sciencedirect.com/science/article/pii/S0263224124013174

### 方法分类

| 类别 | 代表技术 | 特点 |
|------|----------|------|
| **传统信号处理** | SVD, F-K滤波, 小波变换, EEMD | 物理意义明确，参数易调 |
| **字典学习** | DL, K-SVD | 自适应学习原子，稀疏表示 |
| **低秩稀疏分解** | RPCA, GoDec, WNNM | 低秩背景+稀疏目标分解 |
| **深度学习** | CNN, GAN, Transformer | 端到端，需大量训练数据 |

### 各方法优劣对比

**传统方法**:
- ✅ 计算快，可解释性强
- ✅ 无需训练数据
- ❌  handcrafted features
- ❌ 参数需人工调优

**字典学习**:
- ✅ 自适应数据驱动
- ❌ 字典训练耗时
- ❌ 对新场景泛化有限

**低秩稀疏方法**:
- ✅ 数学理论完善
- ✅ 适合线性/相干杂波
- ❌ 迭代优化计算量大
- ❌ 参数敏感

**深度学习方法**:
- ✅ 端到端，效果通常最好
- ✅ 自动特征学习
- ❌ 需大量标注数据
- ❌ 泛化性差 (跨场景)
- ❌ 黑盒，难解释

### 关键挑战与未来方向
1. **复杂场景适应性**: 多种杂波类型并存
2. **实时处理**: 嵌入式/边缘设备部署
3. **小样本学习**: 减少标注依赖
4. **可解释性**: 深度学习方法的物理意义
5. **多模态融合**: GPR + 其他传感器

def calculate_scr(data, signal_mask, clutter_mask):
    """信杂比"""
    signal_power = np.mean(data[signal_mask]**2)
    clutter_power = np.mean(data[clutter_mask]**2)
    return 10 * np.log10(signal_power / clutter_power)

def calculate_if(scr_before, scr_after):
    """改善因子 (dB)"""
    return 10 * np.log10(scr_after / scr_before)
