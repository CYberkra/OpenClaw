## 💡 经验教训

1. **Turix 使用建议**:
   - 提供明确的 URL 优于模糊描述
   - 简单点击任务成功率更高
   - 多窗口环境需要特别注意

2. **Git 大文件处理**:
   - git-filter-repo 是清理历史 secrets 的有效工具
   - Git Credential Manager 比明文 Token 更安全

3. **自动化配置**:
   - 分频率（日/周/月/季度）管理不同任务更清晰
   - HEARTBEAT.md 作为任务清单很实用
