## 实现功能

### 核心类: `SVDBackgroundSuppression`

| 方法 | 功能 |
|------|------|
| `decompose()` | SVD分解 |
| `suppress_background(rank)` | 背景抑制 (去除前rank个奇异值) |
| `low_rank_approximation(rank)` | 低秩近似 (保留前rank个奇异值) |
| `auto_select_rank(threshold)` | 自动选择秩 (10%阈值法) |
| `difference_spectrum(rho)` | 差分谱法选择秩 |
| `plot_singular_values()` | 绘制奇异值分布 |

## 关键代码示例

```python
from svd_background_suppression import SVDBackgroundSuppression

# 初始化
processor = SVDBackgroundSuppression(data_matrix)

# 背景抑制 (去除rank=1的背景)
output, background = processor.suppress_background(rank=1)

# 自动选择秩
auto_rank = processor.auto_select_rank(threshold_ratio=0.1)
```
